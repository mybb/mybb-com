MyBB Discord Server

MyBB Staff are seperated in the channel by a special username colour, and are probably around to talk to as well as other MyBB users.

You must not disclose vulnerabilities in MyBB within the Discord Server. Instead, please see the Security Research page for information on how to responsibly disclose vulnerabilities.

Please also see the MyBB Discord Server Rules page which documents the etiquette that you must follow in the channel.
