MyBB Documentation

Contribute to the MyBB Documentation using GitHub

MyBB uses a GitHub repository to manage the official MyBB Documentation, allowing MyBB users and staff to easily collaborate and contribute to the documentation. To make contributions, you just need to fork the docs.mybb.com repository, add or update pages, push the changes to your forked repository, then send a pull request for review. Once the pull request is approved, your changes will be merged into the official MyBB Documentation.

MyBB Website

Contribute to the MyBB Website using GitHub

MyBB uses GitHub Pages to host the MyBB Website for free, allowing the MyBB Community and staff to maintain it. To contribute you just need to fork the mybb.com repository, make your changes, push the changes to your forked repository and send a pull request. Once your pull request is approved it will be merged into the official MyBB Website.
