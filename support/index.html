International Support

Various third-party communities exist for international users to receive support in their language. You can find a list of them at the Community Forums.

MyBB Documentation

Find information about all MyBB features and common problems

The MyBB Documentation is the first place to go for help with MyBB's features. The documentation includes solutions to common problems and a list of all the MyBB features and how to use them.

Visit the MyBB Documentation →

MyBB Community Forums

Ask MyBB staff and other users to assist you

If you can't find a solution to your problem on the MyBB Documentation, you can ask for help on the MyBB Community Forums. Be sure to search for an answer before posting your question. When posting your question, please follow the instructions to receive support more quickly.

Visit the MyBB Community Forums →
